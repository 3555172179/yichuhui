## Image Guidelines

1. **File Formats**: Use JPG for photos, PNG for graphics with transparency
2. **File Sizes**: Compress images to keep file sizes under 200KB when possible
3. **Naming**: Use lowercase with hyphens (e.g., `walk-in-freezer.jpg`)
4. **Alt Text**: All images should have descriptive alt text in the HTML
5. **Responsive**: Consider providing multiple sizes for responsive loading

## Image Optimization

Before adding images, optimize them using:
- TinyPNG or Compressor.io for compression
- Adobe Photoshop or GIMP for resizing
- WebP format for modern browsers (optional)

## Licensing

Ensure all images have appropriate commercial use licenses:
- Use original photography of your products
- Purchase stock photos from reputable sources
- Use Creative Commons licensed images when appropriate
